    HTML 4:
    Layout tag: table, div
    Typographic tag: h1 -h6, p, b, i, img, etc..
    Table structure: table, tr, td
    

    HTML5:(html5 = html4 + newly introduced semantic tags)
    Layout tag: section, headerm footer, main, aside, article, nav
    Typographic tag: strong, em, svg, canvas, del
    Table layout: table, thead, tbody, tr, th, td
